На странице может присутствовать несколько элементов с одинаковым классом. В этом случае стилевые правила будут применены ко всем элементам.

У нас здесь два класса. Для одного элемента допустимо указывать несколько классов в атрибуте class. В этом случае классы перечисляются в атрибуте через пробел. А элементу присваиваются все стили с обоих классов.

Правила именования классов и идентификаторов

● выбирайте имя, которое будет понятно по смыслу любому другому верстальщику (или вам самому, спустя какой-то период паузы работы над проектом).
● имя должно отражать логическое назначение элемента или применяемых к нему стилевых правил.
● имя должно быть как можно более общим и по возможности коротким.
● если имя состоит из нескольких слов, то их возможно разделять либо через тире, либо через нижнее подчеркивание, либо в стиле camelCase. при этом необходимо придерживаться выбранного стиля на протяжении. всего стилевого файла.

Заголовок: Общий селектор смежных элементов

h2 ~ .description {}, сработает для всех элементов с классом description, которые находятся на одном уровне иерархии с h2

h2+.description {} сработает только для .description, который идет сразу за h2
